.. _spinhx:

==================================================
SPINHX Kullanarak GitHub'da Web Sayfası Hazırlamak
==================================================

GitHub'da web sayfaları barındırabileceğinizi biliyor musunuz? Spinhx'i de kullanabilirsiniz!

GitHub, projeler için web sayfaları ve kişisel sayfalar oluşturmanız için imkan tanır. Kişisel sayfanız, kullanıcı adınızla aynı olması gereken özel bir veri havuzunda (repository) tutulur.
