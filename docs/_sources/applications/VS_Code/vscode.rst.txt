.. _vscode:

=============================================
VS Code Editörü ile .rst Dökümanı Hazırlamak
=============================================

VS Code Uygulamasını kullandığınız işletim sistemi için aşağıdaki linkten indirebilirsiniz:

https://code.visualstudio.com/Download

Kurulumu tamamladıktan sonra programı çalıştırınız. ".rst" formatında döküman hazırlayabilmek için öncelikle VS Code uygulamasında "Eklentiler" kısmından aşağıdaki araçları kurmanız gerekmektedir::

  Python
  Preview
  RST Preview
  reStructuredText
  Table Formatter
  Git History
  Code Spell Checker
  reStructuredText Syntax highlight

Visual Studio Code'da GitHub "pull request" yapmanıza ve sorunlarını gözden geçirmenize, yönetmenize olanak tanıyan eklentiyi de kurabilirsiniz:

GitHub pull requests and issues

Vim editörünü kullanmayı tercih ediyorsanız Vim eklentisini de kurabilirsiniz.
