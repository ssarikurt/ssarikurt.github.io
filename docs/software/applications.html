TRUBA üzerinde hangi yazılımlar kurulu?¶

TRUBA üzerinde kurulu olan yazılımların listesine aşağıdaki komut ile ulaşabilirsiniz:

module avail

TRUBA kullanıcı hesabınzıdaki aktif tanımlı modül bilgilerine ise aşağıdaki komut yardımıyla ulaşabilirsiniz:

module list

TRUBA sistemi üzerindeki kurulu kütüphaneleri ve/veya yazılımları kullanabilmek için ilgili modülleri kendi kullanıcınız üzerinde yüklemeniz gerekmektedir. Bu işlemi aşağıdaki komutu slurm dosyanıza yazarak yapabilirsiniz:

module load yazılım-adı
