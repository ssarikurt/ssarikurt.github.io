Sphinx Kurulumu¶

Sphinx, anlaşılabilir ve güzel dökümantasyon oluşturmayı kolaylaştıran bir araçtır. Temel olarak, kod yazıyormuş gibi belgeler yazabilirsiniz ve kodlaması kolaydır. Sphinx web sayfasından ayrıntılı bilgilere ulaşabilirsiniz.

Öncelikle Sphinx’i kurmanız gerekmektedir. Kurulumu iki şekilde yapabilirsiniz:

  1. İzole edilmiş yerel kurulum (xConda ile)

  2. İzole edilmemiş yerel kurulum
