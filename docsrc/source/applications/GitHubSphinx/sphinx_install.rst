Sphinx Kurulumu
===============

Sphinx, anlaşılabilir ve güzel dökümantasyon oluşturmayı kolaylaştıran bir araçtır. Temel olarak, kod yazıyormuş gibi belgeler yazabilirsiniz ve kodlaması kolaydır. `Sphinx web sayfasından <https://www.sphinx-doc.org/>`_ ayrıntılı bilgilere ulaşabilirsiniz.

Öncelikle Sphinx'i kurmanız gerekmektedir. Kurulumu iki şekilde yapabilirsiniz:

1) :ref:`İzole edilmiş yerel kurulum (xConda ile) <sphinx_conda>`

2) :ref:`İzole edilmemiş yerel kurulum <spinhx_noconda>`
