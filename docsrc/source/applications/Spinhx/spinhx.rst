==================================================
SPINHX Kullanarak GitHub'da Web Sayfası Hazırlamak
==================================================

GitHub'da web sayfaları barındırabileceğinizi biliyor musunuz? Spinhx'i de kullanabilirsiniz!

GitHub, kişisel sayfalar ve projeleriniz için web sayfaları oluşturmanıza imkan tanır. Kişisel sayfanız, kullanıcı adınızla aynı olması gereken özel bir veri havuzunda (repository) tutulur. Bu kişisel web sayfası varsayılan olarak https://kullanciadi.github.io'da yer alacaktır. İstediğiniz bir alan adı yönlendirmesine de sahip olabilirsiniz. Proje web sayfaları, varsayılan olarak, https://kullaniciadi.github.io/repository şeklinde sunulur; burada "repository" projeniz için oluşturduğunuz veri havuzunun ismidir.
